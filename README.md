## 🚀 这是什么？

StarOS 是一个基于 **Arch Linux** 的、面向 **开发者、安全研究员和隐私爱好者** 的开源操作系统发行版。

我们并非另一个简单的“定制版”，而是一个 **精心预配置的Arch体验** 。我们为你做好了所有繁琐的初始设置，但你依然拥有Arch Linux无与伦比的**定制自由**和**滚动更新的前沿体验**。

## 🛡️ 核心哲学

*   **安全即基础：** 默认的加密、防火墙和沙盒环境，为你的数字生活筑起高墙。
*   **开箱即用的工作站：** 预置完整的开发工具链和安全工具集，开机即可投入创造。
*   **更懂你的本土体验：** 无缝的中文支持、国内软件源和本土应用兼容性。
*   **极致的纯净与力量：** 基于Arch Linux，保持轻量、纯净与高度可定制。
